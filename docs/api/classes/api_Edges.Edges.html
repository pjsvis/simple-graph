    Class Edges

    Manages CRUD operations for edges in the graph database. Provides methods to get, add, update, and delete edges, as well as query edges for a specific node.

    Example usage:

    const edges = new Edges(connection);
    await edges.add({ source: 'A', target: 'B', properties: { type: 'references' } });
